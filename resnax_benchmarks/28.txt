// natural language
R1: compulsory, it is a natural number
R2: compulsory, it is in range from ""1"" to ""6""
R3: compulsory, it must be a natural number
R4: optional, it is a natural number

// visual format
concat(concat(<A>,concat(R1,concat(<B>,concat(R2,<_>)))),concat(R3,concat(<_>,R4)))

// example
"A111B2_3_.exe",+
"A111B2_3_0.exe",-

A[0-9]+B[1-6]_[0-9]+_[1-9]*\.exe
